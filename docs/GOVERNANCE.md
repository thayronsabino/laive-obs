# Governança do Projeto

## Visão
Democratizar a transmissão multiplataforma, tornando a tecnologia acessível,
principalmente para igrejas e organizações sem fins lucrativos.

## Mantenedor
O projeto é mantido por Thayron Sabino, responsável pelas decisões finais.

## Contribuições
- Contribuições são bem-vindas via pull requests.
- Todas as contribuições passam por revisão.
- O mantenedor pode solicitar alterações ou recusar contribuições.

## Releases
- Apenas o mantenedor publica versões oficiais.
- Forks são permitidos para estudo e contribuição.

## Código de Conduta
O projeto adota um ambiente respeitoso, colaborativo e inclusivo.

## Licenciamento
O projeto utiliza uma licença de código disponível com restrições comerciais.
Não é considerado open source conforme definição da OSI.
